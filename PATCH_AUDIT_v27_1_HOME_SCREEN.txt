Patch v2.27.1 - Home Screen Start Flow

สิ่งที่ทำ:
1. เพิ่มหน้าแรกตามแบบที่ผู้ใช้เลือก
   - หัวข้อพร้อมวัดคุณภาพแตงโม
   - ภาพแตงโม + คลื่นเสียง
   - ปุ่มใหญ่ เริ่มวัดเสียงเคาะ
   - การ์ดแนะนำเคาะ 3 ครั้ง
   - legend ระดับความสุก: ดิบ/ใกล้สุก/สุกพอดี/อาจเน่า-สุกเกิน
   - ปุ่มประวัติ/วิธีใช้/ตั้งค่า

2. เพิ่ม flow หน้าแรก -> หน้าผลลัพธ์
   - กดเริ่มวัดที่หน้าแรกแล้วซ่อน homeScreen
   - แสดง resultScreen
   - เรียก resetAll() และ startSession() ตามลำดับ

3. คงระบบวิเคราะห์เสียงเดิม
   - ไม่เปลี่ยน trigger, feature extraction, prediction model, voice summary
   - คง id และ event หลักของหน้าผลลัพธ์ไว้

4. ตรวจสอบ
   - JavaScript syntax ด้วย Node
   - runtime smoke test แบบจำลอง DOM

5. ตรวจซ้ำ: ปรับ startFromHome เป็น async เพื่อให้ flow เริ่มวัดรอ startSession ได้ถูกต้องในการทดสอบและใช้งานจริง
