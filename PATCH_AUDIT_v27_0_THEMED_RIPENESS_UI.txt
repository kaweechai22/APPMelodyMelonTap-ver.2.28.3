Patch v2.27.0 - Themed Ripeness UI

สิ่งที่ปรับ:
1. ปรับหน้าตาให้ใกล้แบบตัวอย่าง 4 สถานะมากขึ้น
   - Hero card แบบ 2 คอลัมน์: รูปแตงโมซ้าย ข้อมูลระดับการสุกขวา
   - ป้ายหัวข้อ "ระดับการสุก" ในการ์ดขวา
   - ปุ่มเคาะอีกครั้งเป็นปุ่มวงกลมลอยบนรูปแตงโม
   - ปุ่มเริ่มวัดยังอยู่ด้านบนตามที่ผู้ใช้ต้องการ

2. ปรับสีตามระดับการสุก
   - ดิบ = เขียว
   - ใกล้สุก = เหลือง
   - สุกพอดี = แดง
   - สุกเกิน = เทาอ่อน
   - อาจเน่า = เทาเข้ม

3. ปรับรูปแตงโม
   - ใช้ CSS illustration: แตงโมทั้งลูก + ชิ้นแตงโมผ่า
   - สีเนื้อและสีวงแหวนเปลี่ยนตามระดับการสุก

4. คงระบบวิเคราะห์และ event เดิม
   - id สำคัญยังคงเดิม
   - updateTapProgress รองรับการแสดงผล 2 จุด
   - โค้ด JS syntax ตรวจแล้วผ่าน
